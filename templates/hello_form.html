How it works:

This calculator takes your save and finds your total souls available
(including the souls you would get after ascending, if you leave that box checked)
and calculates the highest optimal levels that you can afford for each of your build's main ancients,
according to the math by Sugima.

Currently, this calculator supports idle and hybrid playstyles. Active builds are not yet considered.
See this post for the differences.

This is not intended as an early-game calculator. It is currently intended for use post-transcension once you have most of these ancients.
